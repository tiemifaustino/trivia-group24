## üíª Run Locally

Clone the project

```bash
  git clone git@github.com:tiemifaustino/trivia-group24.git
```

Go to the project directory

```bash
  cd trivia-group24
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
